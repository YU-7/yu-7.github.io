Windows OpenSSH 登录

2026-05-07 · 1 分钟阅读

安装OpenSSH

Get-WindowsCapability -Online | Where-Object Name -like 'OpenSSH*'
# 添加sshserver
Add-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0
# 启动服务
Start-Service sshd
# 设置开机自启
Set-Service -Name sshd -StartupType Automatic
# 确认状态（Running 为正常）
Get-Service sshd

Windows 电源计划修改为永不休眠，不然 ssh 不能随时随地连接。

ssh 公钥登录问题

明明添加的公钥还是不行，这个问题排查了很久，AI很容易瞎编乱造的，还讲的头头是道的，其实大部分的问题都不会那么复杂，使用 ssh -vvv user@host 的日志说明公钥校验不通过，使用密码替代，其他的信息AI分析不出来的。千问比豆包强点，最后还是GPT的分析是对的，但是也不是一击必中，是我反复强调连接Windows的Openssh。最后发现是Windows的问题，管理员账户的证书存放路径不一样，我放在了自己用户的.ssh目录下，而管理员是要 /c/ProgramData/ssh 。

vim /etc/ssh/sshd_config
# public

whoami
net user xxx
vim administrators_authorized_keys
