灵感碎片与随想

#数字生活
使用 Windows 已经很多年了，乔布斯说过微软最大的问题就是没有品味，我也渐渐明白了这点，但是之前 Apple 给我的感觉就是贵，而且 Apple 经常被调侃“祖宗之法不可变”，在库克的时代都比较保守，但是之前 openclaw 兴起，让我意识到 macmini 的价格其实还是可以接受的，所以我就入手了，当然现在 macmini 的价格已经不值得了。

说 macos 比 Windows 稳定的就是胡扯了，其实都差不多。macos 主要就是 UI 好，有一套统一的逻辑，不像是 Windows 11 的页面还能有 xp 时代的 UI。各有利弊吧，Windows 强大的兼容性让一些几十年前的软件都能运行在最新的 Windows 上，而 macos 就恰恰相反，出了名的不向后兼容，也是 Apple 被人觉得傲慢的一个原因。
在Windows下安装开源的软件非常方便，但是macos就不行，Apple要求需要有软件签名，没有签名就不让你装，虽然有办法绕过去。

Windows的开放策略让很多其他厂商都加入Windows的生态，外设也是，macos的外设实在少（不仅仅是键盘鼠标显示器也是），虽然很多外设都主打双系统，但是使用体验上不咋地。没有经过 Apple 认证的外设，蓝牙连接总是会出现卡死的情况。还有一点就是 macos 的键位和 Windows 很不一样，大部分支持双系统都是以 Windows 为主要考虑对象的，本质上 macos 和 Windows 的键位就不能互存。
